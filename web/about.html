集团概况

以杏仁加工为核心，布局“产品加工 + 品牌连锁 + 文旅 IP”三大板块的综合性集团。

一、集团定位

云烨青集团是一家围绕杏仁加工产品为核心的综合性产业集团，致力于将传统农产品 与现代食品工业、品牌运营及文化旅游深度融合，打造具有区域特色与全国影响力的杏仁产业平台。

集团总部规划选址于辽宁省朝阳市，总部占地面积预计达到 5000 m²， 规划建设现代化生产车间、研发中心、品牌展示与体验空间及文旅互动区域。

基础规划数据

  • 总部位置辽宁省朝阳市
  • 占地面积约 5000 m²
  • 生产线规划2 条智能一体生产线
  • 预计日产能48 万瓶
  • 计划融资规模2300 万人民币

以上数据为项目规划指标，用于对外展示与招商沟通。

二、业务结构与发展路径

阶段一：项目筹备与基础建设

完成项目论证、融资筹措与总部选址，启动厂房与基础设施建设。

阶段二：生产线导入与产品开发

引进 2 条智能一体杏仁加工生产线，开展核心产品与配套系列产品研发。

阶段三：品牌连锁与渠道搭建

以“云烨有礼”为核心品牌，布局礼品渠道、线下体验店与加盟体系。

阶段四：文旅融合与 IP 打造

打造杏仁主题文旅路线、体验馆及城市礼物 IP，形成可持续的文旅消费场景。

三、三大子公司协同布局

云烨青集团通过三家子公司的专业分工，实现“制造 + 品牌 + 文化”的立体化布局：

  • 云烨有礼股份有限公司：礼品与伴手礼品牌运营主体，负责品牌 IP 设计、礼品产品开发、渠道建设与加盟体系搭建。
  • 云烨青食品有限公司：食品制造与研发主体，承担杏仁原料加工、产品品质管控与新品迭代升级。
  • 云烨青文化有限公司：文化与文旅内容运营主体，围绕杏仁文化打造体验项目、文创产品与跨界合作。

制造驱动

依托现代化生产线，确保产品稳定、高效与安全。

品牌驱动

打造可识别、有温度的“云烨”系列品牌形象。

文化驱动

以文化与故事为纽带，提升产品附加值与长期影响力。

四、愿景与使命

企业愿景

成为中国具有代表性的杏仁产业创新平台，通过产品与文化的结合，让一颗杏仁走进更多家庭与城市。

企业使命

把健康、安心与美好体验融入每一瓶产品，让消费者通过一份礼物感知地方文化与生活品质。

发展理念

坚持“品质为本、创新为核、文化为魂”的发展理念，以长期主义视角打造可持续的产业生态。
