杏仁智能加工中心

从一颗杏仁到一瓶高品质产品，全流程智能监控与质量追溯。

三大子公司协同驱动

  • 云烨有礼股份有限公司
  • 云烨青食品有限公司
  • 云烨青文化有限公司

项目定位：面向未来的杏仁产业综合平台

云烨青集团以“产品 + 品牌 + 文旅 IP”为核心驱动力，围绕杏仁这一特色产业， 构建从原料采购、生产加工、渠道拓展，到品牌运营、文旅场景打造的全链路生态。

集团总部选址于辽宁省朝阳市，依托当地区位优势、物流条件与农产品基础， 打造区域标杆性的杏仁产业集群与休闲产业示范项目。

智能化生产

引进 2 条智能一体生产线，实现生产过程自动化、可视化、数据化。

品牌连锁加盟

围绕“云烨有礼”礼品品牌，布局线下体验店与加盟体系。

文旅 IP 打造

以杏仁文化为核心，打造朝阳在地文旅与特色城市伴手礼 IP。

三大子公司职责分工

云烨有礼股份有限公司

专注“礼品+伴手礼”领域，打造线上线下一体化的主题礼品品牌矩阵。

  • 定位“朝阳好礼·云烨有礼”区域伴手礼品牌
  • 开发高颜值、高品质礼盒产品
  • 布局城市礼品店与渠道合作

云烨青食品有限公司

负责杏仁深加工与日常食品研发生产，是集团的制造与品质中枢。

  • 建设智能化杏仁加工生产线
  • 开发饮品、休闲零食、营养食品等系列产品
  • 建立严苛的质量与溯源体系

云烨青文化有限公司

聚焦文化内容打造与文旅 IP 运营，为产品与品牌赋能。

  • 策划杏仁主题文旅线路与体验空间
  • 打造 IP 形象与故事内容矩阵
  • 开展品牌活动与跨界联名合作
